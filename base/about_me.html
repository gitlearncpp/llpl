about me

I decided to pursue my old dreams and learn to program in C and C++.
For several years, I learned other programming languages, but the fascination with C and C++ never faded away.
I decided to create a website dedicated to these languages to, on one hand, motivate myself to work systematically and, on the other, describe interesting elements of these languages.

I hope you will have as much fun on this website as I do.
